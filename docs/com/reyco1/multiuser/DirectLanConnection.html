connect()method 
public function connect(connectionPortId:String = null):void

Establishes a direct connection. If the connectionPortId is privided, then the connection will be established over wifi the other instance of DirectLanConnection on that port. If the argument is not provided, then this instance will be connected to a random port and the "port" property will be set.
